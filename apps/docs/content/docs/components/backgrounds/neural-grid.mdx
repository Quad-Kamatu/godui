edges — the "this product is intelligent" backdrop you see behind AI launches.
Rendered on one lightweight Canvas. Drop it as the first child of a `relative`
container.

## Usage

```tsx
import { NeuralGrid } from "@/components/godui/neural-grid";

<div className="relative overflow-hidden">
  <NeuralGrid />
  <YourContent />
</div>
```

The layer is `absolute inset-0 z-base`; parent must be `relative`, content at
`z-raised`+. Color defaults to `--color-primary` (tracks light/dark). Pauses
off-screen and on hidden tabs; renders a static lattice under
`prefers-reduced-motion`.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `nodeCount` | `number` | `48` | Approximate node count (lattice is sized to match). |
| `pulseSpeed` | `number` | `1` | How fast pulses travel an edge. |
| `color` | `string` | `--color-primary` | Node / edge / pulse color. |
| `density` | `number` | `0.5` | Pulse frequency, `0`–`1`. |
| `nodeSize` | `number` | `2` | Node radius (px). |

Also accepts standard `div` attributes (e.g. `className`, `style`).
