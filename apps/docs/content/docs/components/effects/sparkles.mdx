## Installation

<ComponentInstall componentName="Sparkles" />

Particles are seeded from the canvas area and twinkle by oscillating their
opacity on a single `requestAnimationFrame` loop, resizing with a
`ResizeObserver`. The color string is resolved against the live theme, so
`var(--primary)` tracks light and dark. Under `prefers-reduced-motion` the field
renders a single static frame.

## Usage

```tsx
import { Sparkles } from "@/components/godui/sparkles";
```

```tsx
<Sparkles className="flex h-52 items-center justify-center">
  <h2>Magic</h2>
</Sparkles>
```

## Props

| Prop      | Type     | Default          | Description                                    |
| --------- | -------- | ---------------- | ---------------------------------------------- |
| `color`   | `string` | `var(--primary)` | Particle color.                                |
| `density` | `number` | `30`             | Particles per 10,000 px² of canvas area.       |
| `minSize` | `number` | `0.6`            | Minimum particle radius in pixels.             |
| `maxSize` | `number` | `1.6`            | Maximum particle radius in pixels.             |
| `speed`   | `number` | `1`              | Twinkle speed multiplier.                      |

`Sparkles` also forwards every standard `<div>` attribute to the root element.
