## Installation

<ComponentInstall componentName="ThemeToggle" />

On click, `ThemeToggle` flips the `dark` class on `<html>` inside a
[View Transition](https://developer.mozilla.org/en-US/docs/Web/API/View_Transitions_API),
then animates a `clip-path` circle outward from the cursor so the next theme
wipes into view. Browsers without the API — and visitors with
`prefers-reduced-motion` — get an instant, correct swap instead.

## Usage

```tsx
import { ThemeToggle } from "@/components/godui/theme-toggle";
```

```tsx
<ThemeToggle />
```

### React to changes

`onThemeChange` fires with the next theme after each toggle.

```tsx
<ThemeToggle onThemeChange={(theme) => console.log(theme)} />
```

### Slower reveal

```tsx
<ThemeToggle duration={900} />
```

## Props

| Prop            | Type                                | Default | Description                                  |
| --------------- | ----------------------------------- | ------- | -------------------------------------------- |
| `duration`      | `number`                            | `500`   | Duration of the circular reveal in ms.       |
| `onThemeChange` | `(theme: "light" \| "dark") => void`| —       | Called with the next theme after a toggle.   |

`ThemeToggle` also forwards every standard `<button>` attribute to the element.
