## Usage

```tsx
import { Pagination } from "@/components/godui/pagination";
```

```tsx
const [page, setPage] = useState(1);

<Pagination total={20} page={page} onPageChange={setPage} />;
```

Use it uncontrolled with `defaultPage`, or controlled with `page` + `onPageChange`.
The active-page indicator slides between numbers and long ranges collapse with ellipses.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `total` | `number` | — | Total number of pages |
| `page` | `number` | — | Controlled current page (1-based) |
| `defaultPage` | `number` | `1` | Uncontrolled initial page |
| `boundaryCount` | `number` | `1` | Pages shown at each edge |
| `siblingCount` | `number` | `1` | Pages shown on each side of the current |
| `onPageChange` | `(page: number) => void` | — | Called when the page changes |
