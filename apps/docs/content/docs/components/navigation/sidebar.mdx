## Usage

```tsx
import { Sidebar } from "@/components/godui/sidebar";
```

```tsx
const items = [
  { id: "home", label: "Home", href: "/", icon: <HomeIcon /> },
  {
    id: "projects",
    label: "Projects",
    children: [{ id: "godui", label: "GodUI", href: "/p/godui" }],
  },
];

<Sidebar
  items={items}
  activeId={activeId}
  header="GodUI"
  onNavigate={(id, href) => router.push(href)}
/>;
```

The rail expands on hover, the active item carries a sliding accent bar and
highlight, and nested groups expand with a height spring. Set
`expandOnHover={false}` for a pinned rail with a manual collapse toggle.

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `items` | `SidebarItem[]` | — | `{ id, label, icon?, href?, badge?, children? }` |
| `activeId` | `string` | — | id of the active item |
| `defaultCollapsed` | `boolean` | `false` | Start as an icon rail |
| `expandOnHover` | `boolean` | `true` | Expand the rail while hovered |
| `header` | `ReactNode` | — | Brand node at the top |
| `footer` | `ReactNode` | — | Node pinned to the bottom |
| `onNavigate` | `(id, href?) => void` | — | Called on item click (prevents default) |
