## Installation

<ComponentInstall componentName="Typography" />

## Usage

```tsx
import { Typography } from "@godui/components";
```

```tsx
<Typography variant="h1">Page title</Typography>
<Typography variant="p">Paragraph text.</Typography>
<Typography variant="code">npm install @godui/components</Typography>
```

## Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `variant` | `"h1" \| "h2" \| "h3" \| "h4" \| "p" \| "lead" \| "large" \| "small" \| "muted" \| "code"` | `"p"` | Semantic text style to render |
